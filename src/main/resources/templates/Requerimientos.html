Requerimientos del projecto

Requerimientos Funcionales

    Funcionalidad Descripcion Asociado
    Creacion de cuenta El usuario cliente debe poder crear una cuenta; se le deben solicitar datos necesarios junto conuna casilla final para registrar una vez finalizada la activacion. Cliente
    Activacion de cuenta Al enviar el registro el sistema debe de enviar un mensaje de activacion de cuenta al correo electronico con el fin de comprobar la identidad Cliente
    Autenticacion Una vez hecho el registro el sistema de poder determinar correctamente cuando el usuario ingrese Cliente
    Ver propiedades Una vez dentro el usuario cliente debe poder ver la lista de propiedades que ha creado anteriormente Cliente
    Crear propiedad El cliente debe poder agregar propiedades a la lista de propiedades creadas Cliente
    Editar propiedad El cliente debe poder seleccionar una de las propiedades de su lista y editar los contenidos de la misma Cliente
    Subir foto de propiedad El cliente debe poder subir una foto de como es la propiedad Cliente
    Eliminar(desactivar propiedad) El cliente debe poder eliminar una de sus propiedades, no obstante se tiene que hacer con soft-delete Cliente
    Ver solicitud de arriendo El cliente debe poder ver las solicitudes de arriendo que de arriendo que se han hecho a sus propiedades Cliente
    Aceptar solicitud de arriendo El cliente debe poder aceptar la solicitud de arriendo Cliente
    Cancelar solicitud de arriendo El cliente debe poder rechazar la solicitud de arriendo Cliente
    Calificar arrendatario El cliente debe poder calificar como se comporto el arrendatario del cual acepto la solicitud Cliente
    Creacion de cuenta El administrado debe poder crear na cunta; se deben de solicitar los datos necesarios Admin
    Activacion de usuario Una vez hecha la solicitud se debe enviar un correo de activacion para chequear su identidad Admin
    Autenticacion de usuario Una vez aceptada, este debe poder entrar a la aplicacion con sus credenciales de forma correcta Admin
    Buscar propiedades El admin debera poder buscar propiedades Admin
    Solicitar arriendo Seleccionada una propiedad debe de poder hacer una solicitud para arrendar la propiedad Admin
    Ver solicitudes de arriendo El admin debera de poder ver la solicitudes que ha hecho Admin
    Pagar arriendo El admin debe de poder pagar el arriendo que hizo Admin
    Calificar Finca El admin debe de poder calificar su estadia en la propiedad Admin
    Calificar Arrendatario El admin debe de poder calificar a la persona con la que trato para su estadia Admin
